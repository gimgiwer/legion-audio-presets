# Nahimic IRs and EasyEffects Presets for Lenovo Legion R7000 AHP10

IR profiles recorded from different Nahimic presets on Windows 11 for the Lenovo Legion R7000 AHP10, along with corresponding EasyEffects presets for Linux.

* `irs/` — Nahimic Impulse Response (`.irs`) files. Recorded on Windows 11 at 100% volume with Surround and Volume Stabilizer disabled for a clean EQ curve.
* `presets/` — my EasyEffects (`.json`) configuration files.
